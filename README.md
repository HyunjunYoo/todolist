# 📝 TodoList Project

Vue 3와 Spring Boot를 사용하여 만든 TodoList 애플리케이션입니다.

## 📂 프로젝트 구조
todolist-project/ 
│── frontend/ # Vue 3 프로젝트 
│── backend/ # Spring Boot 프로젝트



## 🚀 **기술 스택**

### **Frontend**
- Vue 3 (Composition API, `<script setup>`)
- Vite
- Vue Router
- Pinia (상태 관리)

### **Backend**
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Lombok
- Gradle (Kotlin DSL)

## 🛠️ **설치 및 실행 방법**

### **1. 백엔드 실행**
```bash
cd backend
./gradlew bootRun

2. 프론트엔드 실행
